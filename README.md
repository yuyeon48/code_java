## ✨ 풀이 규칙 (Commit Convention)

일관성 있는 커밋 메시지 규칙을 사용하여 풀이 과정을 명확하게 기록합니다.

* **커밋 메시지 형식:** `[플랫폼] [난이도] [문제 번호] 문제 이름 / 알고리즘 분류`
    * **예시 1:** `[LeetCode] [Easy] #1 Two Sum / Array, Hash`
    * **예시 2:** `[Programmers] [Lv.2] 타겟 넘버 / DFS/BFS`
* **파일 이름 형식:** `[문제_번호]_[문제_이름].java`
    * **예시:** `1_Two_Sum.java`, `Target_Number.java`
* **코드 주석:** 문제 해결에 사용된 **핵심 알고리즘(자료구조)**, **시간 복잡도** 및 **접근 방식**을 주석으로 간략하게 정리합니다.
